Lưu ý: Chức năng giọng nói hiện đã miễn phí (Edge TTS). Bạn chỉ cần dán API Key nếu muốn sử dụng chức năng Tạo ảnh Postcard.

Hướng dẫn sử dụng

1. Cài đặt API Key

Để sử dụng, bạn cần có Gemini API Key từ Google AI Studio. Nhấn vào biểu tượng chìa khóa ở góc phải trên cùng, dán key vào và lưu lại. Key sẽ chỉ lưu trữ cục bộ trên trình duyệt của bạn.

2. Tạo giọng nói AI

  • Nhập nội dung văn bản bạn muốn chuyển thành giọng nói.
  • Chọn Tone giọng (cảm xúc) và Voice (nhân vật).
  • Điều chỉnh Tốc độ đọc phù hợp.
  • Nhấn Tạo giọng nói AI và chờ kết quả. Bạn có thể nghe thử hoặc tải file âm thanh (.wav) về máy.

3. Tạo ảnh Postcard

Nhấn Tạo ảnh Postcard để AI tự động vẽ một bức ảnh nền phong cách cinematic, kèm theo đoạn text của bạn hiển thị đẹp mắt bên trên.

4. Ngôn ngữ

Hỗ trợ đổi gợi ý bằng tiếng Việt và tiếng Anh ở góc trên. Bạn có thể nhập prompt bằng bất kỳ ngôn ngữ nào AI cũng hiểu.
